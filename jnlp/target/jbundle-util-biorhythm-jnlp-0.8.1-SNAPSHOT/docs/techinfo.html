Biorhythm - Technical Notes

Biorhythm is a Java2 Application. The key design goals for Biorhythm were:

  • Deployment flexibility. Biorhythm can be deployed as an Applet, a Standalone Application, a Java Web Start Application, or a JavaBean all using the same jar file.

  • Design Simplicity. Although Biorhythm has gotten rather large, it is based on the industry-standard MVC (model-view-controller) architecture with simple base classes.

The question I am asked most is, "How do I calculate a Biorhythm for a particular date and graph it." Well, just follow these steps:

  1. First figure out how many days the date is into the current cycle.
    Remainder of: Days from birthday / days in cycle

  2. Next figure out the strength of the signal (It starts at zero an follows a sine wave to +1, 0, -1, and back to 0)
    Convert the remainder to radians and take the sine.

  3. Plot it! (Okay this is an over simplification, but basically that's it!)
